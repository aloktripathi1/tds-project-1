# Q2: Audio Digit Transcription

## Task

Transcribe the 300-digit audio challenge and submit JSON with the predicted number and target hash.

---

## Approach

Used a Python solver script with Groq Whisper transcription and repair logic.

1. Install dependency: pip install groq
2. Set GROQ_API_KEY in environment.
3. Save challenge audio as audio.mp3 in this folder.
4. Run: python solve_challenge.py <TARGET_HASH>
5. Script writes final output to submission.json.
