## Task

## Approach

Contributed one focused and useful PR in an active open-source repository.

1. Pick active repo with clear contribution rules.
2. Make one tiny useful change.
3. Open one clean PR with only related edits.
4. Respond politely to reviewer comments.
5. Keep PR public after merge for grading.
